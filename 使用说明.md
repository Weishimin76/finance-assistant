# 跨境财务助手Pro - 使用说明

## 📦 文件说明

- **跨境财务助手Pro.exe** - 主程序文件，双击即可运行
- 文件大小：约 350MB（包含所有依赖）
- 系统要求：Windows 10/11 64位系统

## 🚀 快速开始

### 1. 首次使用
1. 双击运行 `跨境财务助手Pro.exe`
2. 首次使用需要设置密码（请牢记密码，用于后续登录）
3. 设置完成后，使用密码登录系统

### 2. 日常使用
1. 双击运行程序
2. 输入密码登录
3. 开始使用各项财务功能

## 🔧 功能模块

### 📊 数据看板
- 实时汇率显示（顶部滚动条）
- 财务数据概览
- 多平台数据汇总

### 📁 多平台对账
支持以下平台数据导入：
- Amazon（订单报告、结算报告）
- eBay（交易报告）
- Shopify（订单导出）
- Shopee（财务报表）
- AliExpress（交易记录）

**操作步骤：**
1. 点击"多平台对账"标签
2. 选择对应平台
3. 上传CSV/Excel文件
4. 系统自动解析并显示数据

### 🔍 异常检测
自动识别以下异常：
- 重复订单
- 佣金异常（超过阈值）
- 价格异常
- 缺失数据

### 📈 报表生成
- 一键生成财务报表
- 支持同比/环比分析
- 导出Excel格式
- 自然语言摘要（给老板看的报告）

### 💰 VAT计算
支持10个国家/地区：
- 英国（20%）
- 德国（19%）
- 法国（20%）
- 意大利（22%）
- 西班牙（21%）
- 荷兰（21%）
- 波兰（23%）
- 捷克（21%）
- 美国（各州不同）
- 日本（10%）

### 🧾 报销审核
- 发票OCR识别
- 金额核对
- 日期验证
- 重复报销检测

### 💱 汇率监控
- 实时汇率更新
- 支持多币种转换
- 历史汇率查询

### 📚 政策监控
- 各国税务政策更新
- 跨境电商法规提醒

### 🗂️ 文档归档
- 文件分类存储
- 全文搜索
- 安全备份

## 🔐 安全说明

1. **密码保护**
   - 密码使用SHA256加密存储
   - 密码保存在本地 `data/.secure_config` 文件
   - 不会上传到任何服务器

2. **数据安全**
   - 所有数据存储在本地
   - 不会上传到云端
   - 支持数据备份和恢复

3. **隐私保护**
   - 无需联网即可使用核心功能
   - AI问答功能需要连接Ollama本地服务

## ⚠️ 注意事项

1. **首次设置密码后请牢记**，忘记密码需要删除 `data/.secure_config` 文件重置
2. **数据文件建议定期备份**，复制整个 `data` 文件夹即可
3. **Ollama AI功能需要额外安装**：
   - 下载安装 Ollama：https://ollama.com
   - 拉取模型：`ollama pull qwen2.5`

## 📞 技术支持

如有问题，请检查：
1. 是否使用Windows 10/11 64位系统
2. 是否有足够的磁盘空间（建议预留1GB）
3. 杀毒软件是否误报（可添加信任）
