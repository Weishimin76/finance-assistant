# AI跨境财务管理平台 v9.1 - 部署指南

## 版本特性
- ✅ 实时联网数据更新（汇率/税率）
- ✅ 22个平台费率支持
- ✅ 防幻觉AI引擎
- ✅ Excel报表导出
- ✅ 可视化图表
- ✅ 多用户数据隔离

## 部署方式一：Streamlit Cloud（推荐）

### 步骤1：准备GitHub仓库
1. 在GitHub创建仓库：`finance-assistant`
2. 使用GitHub Desktop克隆到本地
3. 运行 `部署到GitHub_v9.bat` 复制文件
4. 在GitHub Desktop提交并推送

### 步骤2：部署到Streamlit Cloud
1. 访问 https://share.streamlit.io/deploy
2. 选择GitHub仓库：`你的用户名/finance-assistant`
3. 主文件路径：`app.py`
4. 点击 Deploy

### 步骤3：配置环境变量（可选）
在Streamlit Cloud的Settings > Secrets中添加：
```toml
[ollama]
url = "http://localhost:11434"
```

## 部署方式二：本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```
