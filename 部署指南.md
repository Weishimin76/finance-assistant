## 部署方式一: Streamlit Cloud (推荐)

### 步骤1: 更新GitHub仓库

由于文件较大，建议使用以下方式更新:

#### 方式A: 使用Git命令行
```bash
# 1. 克隆仓库
git clone https://github.com/Weishimin76/finance-assistant.git
cd finance-assistant

# 2. 复制新文件到仓库目录
# 将项目文件夹中的所有.py文件复制到此目录

# 3. 提交更改
git add .
git commit -m "Update to v7.0 - AI增强版 添加滚动知识条"
git push origin main
```

#### 方式B: 手动上传
1. 访问 https://github.com/Weishimin76/finance-assistant
2. 点击每个文件，选择 "Edit this file"
3. 复制新文件内容粘贴进去
4. 提交更改

### 步骤2: 部署到Streamlit Cloud

1. 访问 https://share.streamlit.io/
2. 使用GitHub账号登录 (Weishimin76)
3. 点击 "New app"
4. 选择仓库: `Weishimin76/finance-assistant`
5. 主文件路径: `app_secure.py`
6. 点击 "Deploy"

### 步骤3: 访问应用

部署完成后，您将获得一个类似以下的URL:
```
https://finance-assistant-xxx.streamlit.app/
```

## 部署方式二: 本地运行

### 环境要求
- Python 3.8+
- pip

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动应用
```bash
streamlit run app_secure.py
```

### 访问应用
浏览器打开: http://localhost:8501

## 功能说明

### 1. AI智能经营洞察
- 位置: 上传文件窗口下方
- 功能: 支持自然语言查询财务数据
- 快捷问题: 本月各平台利润分析、异常订单检测、经营报告生成等
- 要求: 需要本地安装Ollama并下载qwen2.5:14b模型

### 2. 跨境财务知识学堂
- 位置: 汇率表下方

- 知识领域: 汇率风险管理、VAT递延清关、转移定价、税务合规等

### 3. 多平台数据对账

- 功能: 自动标准化、异常检测、报表生成

### 4. 专业工具箱
- 全球税费引擎

## AI功能配置

### 安装Ollama
1. 访问 https://ollama.com
2. 下载并安装Ollama
3. 下载模型:
```bash
ollama pull qwen2.5:14b
```
4. 启动Ollama服务

### 验证AI连接
启动应用后，AI对话区域会显示连接状态:
- 🟢 在线: AI功能可用
- 🔴 未连接: 需要安装/启动Ollama

## 安全说明

1. 密码采用SHA256加密存储
2. 所有财务数据仅在本地处理

3. 文件编码是否为UTF-8
