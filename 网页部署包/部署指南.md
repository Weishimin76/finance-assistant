# 跨境财务助手 Pro - 部署指南

## 🚀 方案一：Streamlit Cloud（最简单，免费）

### 步骤：

1. **注册 GitHub 账号**：https://github.com（如果已有可跳过）

2. **创建仓库并上传代码**：
   - 新建仓库，名称如 `finance-assistant`
   - 上传以下文件：
     - app_secure.py
     - config.py, parsers.py, reconciliation.py, reports.py
     - finance_llm.py, vat.py, expense_audit.py
     - exchange_rate.py, tax_policy.py, archive.py
     - requirements.txt
     - .streamlit/config.toml

3. **部署到 Streamlit Cloud**：
   - 访问 https://share.streamlit.io
   - 用 GitHub 账号登录
   - 点击 "New app"
   - 选择你的仓库，主文件填 `app_secure.py`
   - 点击 "Deploy"

4. **获取链接**：部署完成后会得到一个永久链接，如：
   ```
   https://your-app-name.streamlit.app
   ```

5. **发给客户**：直接发这个链接，客户打开就能用！

## 🚀 方案三：自己的服务器

### 使用 Docker：

```bash
# 构建镜像
docker build -t finance-assistant .

# 运行容器
docker run -d -p 8501:8501 --name finance finance-assistant

# 访问
http://你的服务器IP:8501
```

### 直接运行：

```bash
# 安装依赖
pip install -r requirements.txt

# 启动
streamlit run app_secure.py --server.port 8501 --server.address 0.0.0.0
```

## ❓ 常见问题

**Q: 部署后打不开？**
A: 检查 requirements.txt 是否完整上传

**Q: 密码忘记了？**
A: 在 Streamlit Cloud 的 Secrets 中添加：
```
RESET_PASSWORD = "true"
```

**Q: 想修改端口？**
A: 修改 .streamlit/config.toml 中的 port 值
